## 📖 專案簡介 (Project Overview)

**Bella-Shoe** 是一個專注於前端互動體驗的電商介面專案。
本專案實作了完整的電子商務購物流程，從使用者的註冊登入、直觀的商品瀏覽與分類篩選、加入購物車的互動體驗，一直到最後的訂單紀錄查詢。目標是打造流暢且響應迅速的用戶介面。

---

### 2. 商品瀏覽 (Product Browsing)
提供直觀的商品展示佈局，支援格狀排列。使用者可以瀏覽全部商品，或是透過分類功能快速尋找特定款式。

* **首頁 & 全部商品:** 展示所有上架商品，版面整潔清晰。
* **分類篩選:** 例如篩選「跟鞋」類別，快速過濾商品。

### 3. 購物流程 
使用者可以點擊商品查看細節，並將喜歡的商品加入購物車。

* **商品明細 (Details):** 詳細顯示價格、商品描述與規格尺寸。
* **加入購物車 (Add to Cart):** 點擊按鈕後的互動回饋，讓使用者確認商品已加入。
